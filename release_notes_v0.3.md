# AUDIT-FMEA reproducibility artifact v0.3

This release contains the AUDIT-FMEA reproducibility artifact for the manuscript:

**AUDIT-FMEA: Auditable Multi-Agent Workflows for Design Failure Analysis**

Included materials:

- prompts and compact evidence material;
- normalized DFMEA outputs for template-only, single-LLM, RAG-LLM, and AUDIT-FMEA workflows;
- repeated-run and role-ablation outputs;
- metric scripts and generated summary tables;
- Figures 1-3 in SVG, PDF, and PNG formats;
- source data for Figures 1-3;
- manuscript-supporting documentation for data availability, figure contracts, and literature verification.

Boundary:

This artifact supports structural auditability analysis only. It does not provide expert validation, physical braking tests, safety certification, or validated S/O/D scoring.
